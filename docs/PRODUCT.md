# gugu 产品文档

> 自动化 A 股交易系统 · 产品说明

---

## 一、产品定位

**gugu = 给开发者自用的自动化 A 股交易系统**

| 维度 | 说明 |
|------|------|
| 核心功能 | 免费行情采集 → 策略选股 → 自动买卖 → 飞书实时通知 → Web 仪表盘 |
| 目标用户 | 个人投资者（开发者本人），非通用 SaaS |
| 交付形态 | Python 后端服务 + 飞书卡片消息 + Web 仪表盘 |

## 二、解决的痛点

| 痛点 | gugu 的解决方案 |
|------|----------------|
| 没时间盯盘、执行交易纪律 | APScheduler 定时调度，自动采集行情、执行信号、下单 |
| 人工选股易被情绪左右 | 8 个内置量化策略 + 多策略信号融合，纪律性执行 |
| 缺统一风控 | 三级风控（L1 仓位 / L2 日亏 / L3 系统规则），铁律不可绕过 |
| 信号/持仓/盈亏信息分散 | 飞书卡片消息统一推送（信号 / 日报 / 告警 / 回测 / 异常） |
| 策略验证成本高 | 自研回测引擎 + 模拟盘，零资金风险验证策略 |

## 三、五阶段架构（渐进式）

| 阶段 | 名称 | 状态 | 说明 |
|------|------|------|------|
| 1 | 策略固化 | ✅ 已完成 | 自然语言 → Python 策略代码（LLM 生成 + 校验 + 注册） |
| 2 | 回测 | ✅ 已完成 | 历史数据验证策略有效性，输出收益/夏普/回撤/胜率 |
| 3 | 模拟盘 | ✅ 已完成 | 真实行情 + 模拟资金 + 飞书通知，零资金风险 |
| 4 | 小额实盘 | ⏳ 待启用 | QMT 接口 + 小号账户 + 人工确认机制 |
| 5 | 正式实盘 | ⏳ 未来 | 多数据源备份 + 监控告警 + 灾备 + 审计日志 |

## 四、核心功能

### 4.1 策略层

**8 个内置策略**：

| 类型 | 策略 | 说明 |
|------|------|------|
| 趋势跟踪 | Turtle | 海龟突破（20 日高低点） |
| 趋势跟踪 | DualMA | 双均线交叉（5/20 日） |
| 趋势跟踪 | MACD | MACD 柱状图信号 |
| 趋势跟踪 | SAR | 抛物线转向点 |
| 均值回归 | RSI | RSI 超买超卖 |
| 均值回归 | KDJ | KDJ 金叉死叉 |
| 均值回归 | Bollinger | 布林带突破 |
| 突破 | DualThrust | 双突破（N 日高低点） |

**LLM 自然语言策略生成**：描述策略逻辑 → LLM 生成 Python 代码 → AST 校验 → 自动注册。

### 4.2 风控层（三级铁律）

| 级别 | 规则 | 动作 |
|------|------|------|
| L1 单股 | 单只仓位 ≤ 30% | 超限拒绝加仓 |
| L2 日亏 | ≥ 3% 预警，≥ 5% 熔断 | 预警飞书通知，熔断停止交易 |
| L3 系统 | T+1 / 涨跌停 / 停牌 | 自动过滤 |

### 4.3 通知层（飞书卡片）

| 类型 | 触发时机 | 内容 |
|------|----------|------|
| 信号通知 | 策略产生买卖信号 | 股票/方向/策略/理由/建议仓位/当前价 |
| 每日日报 | 09:10 / 11:35 / 15:10 | 市场概览/热门板块/今日信号/持仓概况 |
| 风控告警 | 触发风控规则 | 级别/详情/建议动作 |
| 回测报告 | 回测完成 | 收益/夏普/回撤/胜率 |
| 系统异常 | 采集失败/引擎崩溃 | 异常详情/恢复建议 |

### 4.4 回测引擎

- 输入：历史 OHLCV 数据 + 策略 + 初始资金
- 输出：收益曲线、夏普比率、最大回撤、胜率、盈亏比
- 限制：未接入完整风控（L2/L3），回测结果可能偏高

### 4.5 模拟盘

- T+1 结算（买入当日不可卖）
- 佣金 0.025% / 印花税 0.1%（卖出）/ 滑点 0.2%
- **状态持久化**：JSON 文件存储，进程重启自动恢复持仓和交易记录
- 持仓盈亏实时更新
- CLI 持仓查看：`uv run python scripts/show_portfolio.py --all`

### 4.6 Web 仪表盘

- FastAPI 后端 + 单页 HTML 前端
- 实时展示：总资产、现金、持仓市值、累计盈亏
- 持仓详情：代码/数量/成本/现价/盈亏
- 交易记录：最近 20 笔交易
- 风控状态：日亏比例、熔断状态
- 30 秒自动刷新
- 启动：`uv run python scripts/run_web.py`

## 五、使用方式

### 5.1 环境准备

```bash
# 克隆项目
git clone https://github.com/ClayDong/gugu.git
cd gugu

# 安装依赖（推荐 uv）
uv sync
# 或 pip install -e .

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入飞书 app_id / app_secret / chat_id
```

### 5.2 配置

编辑 `config/settings.yaml`：

```yaml
watchlist: ["600519", "000858"]  # 自选股
auto_select: false                # 自动选股（默认关闭）
initial_capital: 1000000           # 初始资金
```

### 5.3 运行

```bash
# 回测
uv run python scripts/run_backtest.py

# 模拟盘
uv run python scripts/run_paper.py

# Web 仪表盘
uv run python scripts/run_web.py

# 实盘（阶段四）
uv run python scripts/run_live.py
```

## 六、能力边界与已知限制

1. **自动选股默认关闭**：数据源稳定性待验证，建议先手动配置自选股
2. **回测引擎未接入完整风控**：回测结果可能高估
3. **交易日历降级仅覆盖当年**：跨年运行需注意
4. **飞书熔断恢复仅 CLI**：`engine.reset_halt()` 手动调用
5. **降级源仅行情**：SinaCollector 不支持行业/个股资金流明细
