# 泡沫参与决策 (Bubble Participation)

## R — 原文 (Reading)

> 经济史是一部基于假象和谎言的连续剧,经济史的演绎从不基于真实的剧本,但它铺平了累积巨额财富的道路。做法就是认清其假象,投入其中,在假象被公众认识之前退出游戏。
>
> — 陈江挺,《炒股的智慧》第七章 抓住大机会

> 回头想想,人们会嘲笑当年的民众真是疯了。当年也不是没有头脑清醒的人,但他们太早了一步,他们指出这个泡沫会破碎,但市场用不断升高证明他们论断的错误。开始还有人听听他们的警钟,随后便嘲笑他们的短视。
>
> — 陈江挺,《炒股的智慧》第七章

## I — 方法论骨架 (Interpretation)

作者的核心洞见(基于索罗斯反身性理论): 泡沫不是"避免"的,而是"识别+利用"的工具。前提是 **1) 你能识别假象 2) 你的退出纪律强于你的贪心**。

**泡沫参与的三阶段框架**:

**阶段 1: 识别假象**
- 价格已脱离基本面但仍有上涨动力
- 大众从"怀疑"转向"相信"
- 主流媒体开始"正面报道"
- 检查清单:
  - [ ] 是否所有人都在谈论 X 赚钱?
  - [ ] 是否出现"新概念"叙事("这次不一样")
  - [ ] 是否出现"主流机构入场"的报道?
  - [ ] 是否垃圾/边缘标的也开始上涨?(末期信号)

**阶段 2: 投入其中**
- 资金量: 不超过总资金 10-20%(必须能承受全损)
- 入场时机: 在大众"半信半疑"时入场,不要等"全民相信"
- 跟踪指标: 龙头股表现(是否开始疲软)、垃圾股表现(是否开始活跃)
- 心理准备: 准备好"提前退出"——大众觉醒时必须已离场

**阶段 3: 提前退出**
- 退出信号预设: 写出硬规则(例:"X 股票跌破 30 日均线即清仓")
- 不依赖"感觉"——泡沫末期"感觉还会涨"恰恰是反向信号
- 退出时不要再入场——泡沫的"反弹"是诱多
- 退出后: 写复盘——为什么这次判断对/错,下次怎么改进

**核心原则**:
- **不试图逃顶**——泡沫顶不可预测
- **不试图抄底**——泡沫破裂后的"便宜"不是便宜
- **不参与"末期"**——大众已普遍相信时入场胜算急剧下降
- **不超出风险承受力**——泡沫破裂时可能没有流动性

## A1 — 书中的应用 (Past Application)

### 案例 1: 1720 年南海泡沫
- **问题**: 南海公司股票半年内升 8 倍,总市值达全欧洲现金流通量 5 倍
- **方法论的使用**: 这是"假象+参与+退出"的经典案例。早期识别假象(贸易垄断故事不可持续)的人,本可参与后退出
- **结论**: 一个月内从 1000 英镑跌到 129 英镑,无数投资者破产
- **结果**: 接受南海公司股票做抵押的银行一间间倒闭,英格兰银行也仅以身免
- **应用**: 假象破裂的速度(一个月跌 87%)——这就是为什么"提前退出"是硬规则

### 案例 2: 1920 年代佛罗里达地产疯
- **问题**: 迈阿密 75000 人口中有 25000 个地产经纪人,超出 2000 家地产公司
- **方法论的使用**: 1925-1926 年的"全城皆疯狂"——识别"疯狂阶段"的信号
- **结论**: 1926 年泡沫破裂,银行接连倒闭
- **结果**: 笔者(作者本人)80 年代末也加入售卖,30 年前 5000 美元买的土地只能卖 7500 美元,跨越半个世纪仍未恢复
- **应用**: 即使是 30 年后,泡沫破裂后的"便宜"也不是便宜

### 案例 3: 80 年代科威特股疯
- **问题**: 科威特用石油收入托市,股民用远期兑现支票(实际无钱)购买股票
- **方法论的使用**: 50 亿美元膨胀到 1000 亿美元(20 倍泡沫)+ 杠杆
- **结论**: 1982 年交易所危机,泡沫彻底破裂
- **结果**: 科威特金融体系遭受重创
- **应用**: 杠杆放大泡沫也放大破裂——泡沫参与时绝不能加杠杆

### 案例 4: 90 年代永生集团"一灵散"案例
- **问题**: 永生集团"一灵散"从 5 元涨到 400 元,基于"包治百病"假象
- **方法论的使用**: 信息反转(专利不批准→技术员揭露白糖熬制)导致泡沫破裂
- **结论**: 400 → 300 → 150 → 停牌
- **结果**: 公司被停牌,赵主席被捕
- **应用**: 假象破裂的"信息"通常很微小,但能引发连锁反应——必须提前离场

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **资产价格明显脱离基本面**: "X 已经涨了 5 倍,但听说还要涨"
2. **大众普遍参与**: "我身边所有人都在讨论/买 X"
3. **新概念叙事**: "X 是新范式,旧估值不适用"
4. **机构入场报道**: "X 已被主流机构接纳"
5. **想"赚泡沫的钱"**: "我知道是泡沫,但想赚一段"

### 这些情境的语言信号是什么?

- "该不该买""能涨到多少""是泡沫吗"
- "所有人都在买""我邻居/同事/亲戚都在买"
- "这是新范式""旧规则不适用"
- "X 已经涨了 X%"

### 和哪些相邻 skill 不同?

- **vs stock-entry-decision**: 那个是基础+技术+阶段三层过滤,本 skill 是"明知是泡沫仍参与"的特殊决策
- **vs stock-profit-taking-decision**: 那个是"该不该兑现",本 skill 是"该不该进入泡沫 + 何时退出"
- **vs stock-market-timing**: 那个是大市择时,本 skill 是特定泡沫的参与策略

## E — 执行步骤 (Execution)

### 步骤 1: 自我评估——你具备参与泡沫的资格吗?

**动作**: 三个硬性资格检查:
- [ ] 我能否识别泡沫的"假象"特征?(不是事后)
- [ ] 我是否有强制退出规则?
- [ ] 我能否承受全损?

**完成标准**: 三个都是"是"。任一为"否",**拒绝参与**。

### 步骤 2: 评估泡沫阶段

**动作**: 用"疯狂阶段"识别清单:
- [ ] 是否所有人都在谈论 X?
- [ ] 是否出现"新概念"叙事?
- [ ] 是否主流机构入场?
- [ ] 是否垃圾标的也开始上涨?
- [ ] 是否出现"这次不一样"的说法?

**完成标准**: 1-2 个信号=可考虑参与(早期); 3-4 个=谨慎; 5 个=末期,拒绝。

### 步骤 3: 设定硬规则

**动作**: 写出三套硬规则,签字画押:
- **入场规则**: "X 信号出现我才入场"(避免冲动入场)
- **退出规则**: "X 信号出现我立即清仓"(必须有明确信号)
- **资金规则**: "不超过总资金 X%"(必须能承受全损)

**完成标准**: 三套规则都写下来,且**有数字**(不是"看情况")。

### 步骤 4: 跟踪泡沫状态

**动作**: 每周(每天更好)检查:
- 龙头股表现(是否开始疲软)
- 大众情绪(从"半信半疑"转向"全民相信"时,是退出信号)
- 退出规则的信号是否触发

**完成标准**: 用户有跟踪表/日记。

### 步骤 5: 触发退出规则时立即执行

**动作**: 一旦退出规则触发,**24 小时内清仓**,不要"再等等"。

**完成标准**: 用户在触发日清仓。

### 步骤 6: 复盘

**动作**: 退出后,写复盘:
- 这次判断对/错?
- 退出是早了还是晚了?
- 下次怎么改进?

**完成标准**: 用户能用 1-2 段话总结。

## B — 边界 (Boundary) ★

### 什么时候**不要**使用本 skill

1. **用户没识别出泡沫时**: "X 看起来不错,要不要买" → 这是一般入场决策,用 stock-entry-decision
2. **用户有"道德反对"但没"利益权衡"**: "泡沫是不对的,我不参与" → 不需要决策框架,直接拒绝即可
3. **用户无法承受全损**: 失业/借贷/救命钱参与泡沫,直接拒绝
4. **用户价值投资长线持有**: 与泡沫参与逻辑不同
5. **用户在做空/对冲**: 那是另一套框架,不适用本 skill

### 来自书中的警告

- **"经济史是假象+谎言的连续剧"**: 作者本人对泡沫持"工具性"态度,不是道德谴责
- **不试图逃顶**: 泡沫顶不可预测
- **不试图抄底**: 泡沫破裂后价格"便宜"不是便宜
- **不参与末期**: 大众普遍相信时胜算急剧下降

### 作者盲点

- **作者以索罗斯/历史案例为主,未充分讨论中国 A 股的特殊性**——A 股的涨跌停板、T+1 制度使泡沫参与/退出策略不同
- **作者未涉及现代算法交易对泡沫的影响**——程序化交易可能加速泡沫的涨与破
- **作者未涉及加密资产/数字货币的泡沫**——这是新现象,可能需要新框架

### 与相邻 skill 的边界

- **stock-entry-decision**: 那个是常规入场,本 skill 是泡沫参与的特例
- **stock-profit-taking-decision**: 那个是已浮盈,本 skill 是"决定是否进入浮盈"
- **stock-market-timing**: 那个是大市择时,本 skill 是特定资产的择时
- **stock-psychology-check**: 那个是情绪检查,本 skill 假设用户情绪稳定(否则不应参与泡沫)
