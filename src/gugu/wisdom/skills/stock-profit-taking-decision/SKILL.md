# 止盈决策 (Profit-Taking Decision)

## R — 原文 (Reading)

> 华尔街将炒股的诀巧归纳成两句话:截短亏损,让利润奔跑!英文叫 Cut loss short, let profit run!意思即是一见股票情况不对, 即刻止损, 把它缩得越短越好!一旦有了利润,就必须让利润奔跑,从小利润跑成大利润。
>
> — 陈江挺,《炒股的智慧》第四章

> 炒股高手利物莫特别指明,他炒股的秘诀不是他怎样思考,而是在他买对了的时候能够安坐不动。这是很难的一件事,你要克服对脱手获利的冲动。
>
> — 陈江挺,《炒股的智慧》第三章

---

## I — 方法论骨架 (Interpretation)

止盈决策的核心反直觉: **"卖早了"通常比"卖晚了"更赔钱**。原因是盈亏两端的不对称性——一笔小赚的交易,如果按 50% 概率重复 10 次可能变成 10 次小赚;一笔大赚的交易,如果提前兑现就永远失去放大的可能。

**作者的核心机制: 移动止损代替固定止盈**。不要设"赚 20% 就卖"这种固定规则,而要**随股价上升不断上移止损点(浪谷递进)**。这样保证:
- 如果趋势继续 → 止损被不断上移,锁住更多利润
- 如果趋势反转 → 在移动止损位自动出局,不会从盈利变亏损

**"利物莫安坐不动"原则**: 一旦买入正确,**不要因为"已落袋为安"心理就提前退出**。判断正确=该笔交易有望继续盈利,提前退出=自我背叛。

**止盈的真正时机**: 出现以下三个"危险信号"中的任一个时,立即清仓:
1. 交易量暴增但股价不涨(机构出货信号)
2. 两天转头(连续两日从最高位跌下来)
3. 升势末期出现坏消息(基本面恶化)

**不要做的事**:
- 不要试图寻找最高点(作者原话"你永远不知股票会升多高")
- 不要见"涨了 20%"就跑
- 不要因为"落袋为安"就提前退出

## A1 — 书中的应用 (Past Application)

### 案例 1: 利物莫的"安坐不动"
- **问题**: 利物莫的许多大盈利单都曾"看似应该获利了结",但他没动
- **方法论的使用**: 买入正确 → 安坐不动 → 让利润跑成大利润
- **结论**: 利物莫成为本世纪初最著名的炒家,核心是"买对后的耐心"
- **结果**: 验证了"截短亏损,让利润奔跑"原则

### 案例 2: 微软 + 麦当劳 + 王安的"高增长延长"案例
- **问题**: 这些公司在盈利增长期,股价持续上涨多年
- **方法论的使用**: 投资者如果按"赚 20% 就卖"的固定规则,会错失 13 年百倍的机会
- **结论**: 临界点四阶段中"正常升势"持续时间可达 5-10 年,中间不应退出
- **结果**: 验证了"让利润奔跑"的爆发力

### 案例 3: 永生集团"一灵散"案例 (90 年代中国)
- **问题**: 永生集团"一灵散"从 5 元涨到 400 元,有人以为会继续涨
- **方法论的使用**: 没有在疯狂阶段分批退出,反而被"还会涨"的预期锁定
- **结论**: 信息反转(专利不批准) → 股价跌到 300 → 继续跌到 150 → 停牌
- **结果**: 警示——疯狂阶段应"分批退",不应"贪婪留"

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **已有浮盈**: "我已经赚了 30%,现在该不该卖"
2. **刚盈利**: "X 涨了一点,要不要先落袋为安"
3. **朋友退出后**: "我朋友在 X 上赚了 50% 跑了,我是不是也该跑"
4. **新闻"到顶"**: "新闻说 X 涨太多了要回调,我要不要先卖"
5. **加仓犹豫**: "X 涨了 20%,我想加仓,但又怕是高位"

### 这些情境的语言信号是什么?

- "该不该卖""要不要卖""适合卖吗"
- "已经赚了""赚了点""赚了不少"
- "是不是高位""是不是顶了""会不会跌"
- "落袋为安""先兑现"

### 和哪些相邻 skill 不同?

- **vs stock-stop-loss-decision**: 那个是浮亏,本 skill 是浮盈(决策时情绪相反: 浮亏是不舍得卖,浮盈是急于卖)
- **vs stock-trailing-stop**: 那个是机械化的移动止损技术,本 skill 是决策层(决定什么时候触发)
- **vs stock-entry-decision**: 那个是入场前,本 skill 是持仓中

## E — 执行步骤 (Execution)

### 步骤 1: 阶段判断

**动作**: 评估股票当前处于"临界点四阶段"中的哪一阶段:
- **牛皮** → 评估"是否要进入下一阶段",本 skill 不直接适用
- **正常升势** → **不卖**,移动止损即可
- **疯狂** → **分批退出**(30%/30%/30% 阶梯式减仓)
- **最后** → **立即清仓**

**完成标准**: 用户能正确判断阶段。

### 步骤 2: 检查"危险信号"三个

**动作**: 三个危险信号,任一触发即应减仓/清仓:
- 交易量暴增但股价不涨?
- 两天转头?
- 升势末期出现坏消息?

**完成标准**: 用户逐一检查并给出明确回答。

### 步骤 3: 用反向测试替代"落袋为安"心理

**动作**: 问用户——"**如果今天我没持仓,我会按当前价买入这只股票吗?**"

- 如果答案为**是** → 继续持有(浮盈只是结果,不是退出理由)
- 如果答案为**否** → 减仓(可能阶段已变)
- 如果答案为**不知道** → 减仓到能睡着的仓位

**完成标准**: 用户给出清晰判断。

### 步骤 4: 移动止损机制(替代固定止盈)

**动作**: 教用户用移动止损代替"赚 X% 就卖":
- 入场后设初始止损(浪谷/支撑线/总资金 20%)
- 股价创新高 → 止损移到上一浪谷
- 趋势加速 → 可加码,但不撤止损
- 跌破移动止损 → 自动出局

**完成标准**: 用户写下"我的移动止损规则是 X"。

### 步骤 5: 不要寻找最高点

**动作**: 当用户表达"我想精准逃顶"时,提醒——**没人能持续精准逃顶**。中间一截(60-70% 波幅)即可。

**完成标准**: 用户接受"中间一截"策略,放弃"逃顶幻想"。

## B — 边界 (Boundary) ★

### 什么时候**不要**使用本 skill

1. **用户没盈利时**: "我已经亏了 30%" → 错位信号,转 stock-stop-loss-decision
2. **用户处于"无所谓"状态**: "持有就持有吧" → 不需要决策,不要激活
3. **用户要"逃顶"幻想时**: 本 skill 应拒绝"我想精准逃顶"——逃顶是低期望值行为
4. **价值投资长线持有场景**: 如果用户是基于 3-10 年长期价值持有,本 skill 不适用(用更通用的"再平衡"决策)
5. **指数定投场景**: 定投止盈有专门框架(年化收益率目标),不适用本 skill

### 来自书中的警告

- **贪小便宜是失败主因**: 见 ce03"贪小便宜导致赚小亏大"——这是 p14 失败者模式的核心
- **不要试图寻找最高点**: 试图精准逃顶是低胜算行为
- **不要跟随大流**: 当所有人都在讨论"该不该卖"时,可能恰恰该持有(逆向信号)
- **市场从来不会错**: 你的浮盈不等于"赚到了",只是"浮动的"

### 作者盲点

- **作者对"正常升势"持续时间估计可能偏乐观**——现实中可能仅持续几周就转入"疯狂",需要持续观察
- **"疯狂阶段"识别在信息充分的现代市场更困难**——以往明显的"全城热议"信号可能更分散
- **作者未涉及做空机制**——本 skill 假设只能做多,做空场景下止盈逻辑不同

### 与相邻 skill 的边界

- **stock-stop-loss-decision**: 两者共同构成"截短亏损,让利润奔跑"原则,但触发条件相反(浮亏 vs 浮盈)
- **stock-trailing-stop**: 那是机械执行机制,本 skill 是决策层
- **stock-psychology-check**: 那是识别情绪状态,本 skill 是在情绪稳定后做决策
